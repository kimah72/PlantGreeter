# Overview

I made a greeter with a personality. If I am going to spend a semester working on projects connected to plants, I may as well have one of my own to greet you first. Anna is my oldest plant in my collection and she felt like the perfect ambassador for my software portfolio.

I chose Python as my language because it has been three years since I last used it and it was always my favorite. I barely remember anything when I started. So for me, building a small app that calls on several of Python's core features was the perfect way to learn again — not by following a tutorial, but by figuring it out myself one piece at a time.

Anna will greet you by name, adjust her greeting based on the time of day, and share a piece of plant wisdom that rotates by day of the week. She also has something to say on her way out.

[My Plant Greeter - Hello World](https://www.youtube.com/watch?v=fZIwpkPnCbc)

# Development Environment

- Visual Studio Code
- Python 3
- Git and GitHub for version control

This program was written in Python 3 and uses only the built-in `datetime` module. No additional libraries or installation required.
